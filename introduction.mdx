## Autenticação

Todas nossas APIs são protegidas e para isso você deve ter um token de acesso que deve sempre ser enviado no Header.

### Solicitando token

Para obter seu token de acesso você deve entrar em contato conosco pelo email suporte@probacloud.com informando os dados da sua empresa e enviaremos o seu token em instantes. 

### Utilizando token
> Garanta que sua plataforma esteja se comunicando via API Restful

1- Nome do header: Authorization

2- Conteúdo do header: `seu token de acesso`
